# Proyecto

CONSIDERACIONES DE EJECUCIÓN

A considerar: el archivo proyecto "Proyecto" fue creado en Visual Studio Code.

Para valores no númericos el programa presenta bucles infinitos en el menú, dado que no corresponde el tipo de opción (está descrito como variable numérica).

Para la búsqueda de titulos, deben ser ingresados los parámetros de búsqueda exactamente como es requerido. Ejemplo: Si va a buscar la obra "Chainsaw Man", tiene que escribirlo de esa manera con las letras capitalizadas

Para la busqueda de generos, debe ingresar el genero en minuscula, y si va a hacerlo con dos a la vez, tiene que separarlos con una coma. Ejemplo: "drama,romance" o "accion"

A la hora de ingresar la valoracion de un texto, y quiere ponerle decimales (Como maximo dos) , tiene que poner un punto. Ejemplo: "7.4" o "5.6"
